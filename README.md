
Babycraft
=======
- 本赤石mod基于Neoforge开发，适用于Minecraft 26.1版本 ，其他版本未测试


Features
===========
## 方块
### 幼年方块
- 有概率替换其他方块生成
- 接受随机刻时会成长
- 适配金蒲公英

## 物品
### 金蒲母英
- 可重置方块与生物到幼年状态
- 与金蒲公英有组合技

## 状态效果
### 返老还童
> *谁道人生无再少？*
- 由金蒲公英酿造而成
- 持有此效果的生物会变小
- 以及有一些特殊效果…？


### 重开
- 由金蒲母英酿造而成
- 持有此效果的生物，攻击时有概率使对方陷入返老还童效果
